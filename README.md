# featherweight-vip.github.io

The Featherweight VIP project website — a self-contained static site (plain
HTML + CSS, no build step), served by GitHub Pages.

## Structure

```
index.html              # homepage: hero, value props, protocol×platform matrix,
                        #   what's-in-a-kit, coverage, quickstarts
roadmap.html            # protocol & platform roadmap
guides/
  quickstart.html       # cocotb / UVM / formal quickstart
assets/
  css/site.css          # brand tokens + layout (mirrors the icon palette)
  brand/                # icon, lockup, favicon (synced from ../.github/brand)
favicon.ico
```

## Develop

This site is the org-level front door only. **Per-protocol documentation lives
in each protocol's own repo** (`fwvip-<proto>`); the homepage matrix links out to
them rather than duplicating their docs here.

It's static — just open `index.html`, or serve the folder:

```bash
python3 -m http.server 8000   # then visit http://localhost:8000
```

## Deploy

Push to the default branch of the `featherweight-vip.github.io` repo; GitHub
Pages serves it at `https://featherweight-vip.github.io`. Add a `CNAME` file to
use a custom domain.

## Brand assets

`assets/brand/` is copied from the org's single source of truth in
[`.github/brand/`](https://github.com/featherweight-vip/.github/tree/main/brand)
(generated by `icons/generate.py`). Re-sync when the brand changes; don't edit
the copies here by hand.

Fonts: **Space Grotesk** (wordmark) + **JetBrains Mono** (mono/tagline), loaded
from Google Fonts with system fallbacks.
