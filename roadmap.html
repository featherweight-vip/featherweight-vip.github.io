Roadmap

Two axes: more protocols, and more platforms per protocol. Coverage ships with every kit from day one.

Protocols

Now
Wishbone, SPI — Initiator / Target / Monitor, UVM agent, checkers, coverage.
Next
AXI4-Lite, then AXI4 (multi-stream: read/write channels), AHB, APB, Avalon.
Later
I²C, UART, and community-requested protocols.

Platforms

Now
SystemVerilog / UVM agents; formal checkers under SymbiYosys.
In progress
cocotb signal-level interface to the transactor FIFOs.
Planned
PSS (Accellera Portable Stimulus) + portable coverage; Python operation API over Zuspec/sysmesh DPI FIFOs, with a path to emulation.

Coverage

Coverage is not a milestone — it's part of the kit definition. Today: SystemVerilog covergroups sampled by the Monitor. Planned: a portable coverage model (Zuspec/PSS) so the same intent travels across simulation, formal, and emulation.
